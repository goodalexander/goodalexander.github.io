Powered by the Post Fiat Task Node, this benchmark uses quality of response, cost, and latency through OpenRouter in a 50-25-25 weighting to determine the best chat model for coaching, productivity, and task generation. Dynamically reweights inputs to hot-load a new benchmark. Updated weekly.

Scoring Prompt

The Post Fiat Task node has 20 canonical prompts designed to move users toward their objectives, ranging from task generation, motivation, brainstorming, and visualization. These 20 canonical prompts use real user data along with task and context metadata associated with them. The benchmark works by simulating interactions with these 20 canonical prompts using real user data and live latency output from OpenRouter.

Quality Aggregation

The benchmark quality score is the mean of the three judge scores for each candidate response set. Cost and latency are measured separately, then combined with quality in the public ranking weights.
